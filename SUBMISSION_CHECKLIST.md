## REQUIRED — full 7-seed validation before you submit (15 min)

V21 self-calibrates per seed. It was validated on six seeds in pairs
(mean Delta ~ +0.10, min Delta ~ +0.092), but you must confirm the
aggregate on the exact seed set you submit:

```powershell
python run.py --adapter adapters.myteam:Engine --seeds 7 13 31 97 211 503 1009 --out report.json
Get-Content report.json | ConvertFrom-Json | Select-Object -ExpandProperty aggregated
```

Decision rule:
- min_delta > 0 on every seed AND mean_delta >= 0.076  -> ship V21.
- Any seed with min_delta < 0, or mean below V19's 0.076 -> ship V19
  (`Copy-Item adapters\myteam_v19.py adapters\myteam.py -Force`).

V21's six-seed evidence strongly predicts a clean pass, but the full
run is the gate. Do not skip it.

## Speed note — the precompute cache

V21 memoises its `__init__` precompute to disk (`.pcam_cache/`). The
first run of each seed takes ~60-90s; every repeat run of the same seed
is instant. Iterate freely — only the first pass is slow. The cache is
keyed by a hash of patterns + R + params, so a fresh seed always
recomputes. Force a clean recompute by deleting `.pcam_cache/` or
setting `$env:PCAM_NO_CACHE=1`.
